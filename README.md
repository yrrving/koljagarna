# Koljagarna

En spelbar klassrumsprototyp for hogstadiets NO-undervisning om kolcykeln.

Spelet ar en duell mellan Lag Utslapp och Lag Lagring. Utslappssidan far
snabba poang genom aktiviteter som slapper ut kol, medan lagringssidan bygger
projekt som vatmarker, skog och jordhalsa som ger effekt over tid.

Oppna `index.html` direkt eller publicera mappen via GitHub Pages.
